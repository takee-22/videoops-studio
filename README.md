# VideoOps Studio

A simple desktop video utility app built with PySide6 and FFmpeg.

## Features

- Visual split with cut points
- Split into 2 or multiple pieces
- Merge multiple videos
- Smart merge output naming
- Convert DAV/MP4/MKV/MOV/AVI/MPEG
- Safe mode for DAV exports

## Run

```cmd
python -m videoops_studio.app
